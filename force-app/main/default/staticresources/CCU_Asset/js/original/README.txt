In this project Deloitte Digital has restyled the following tools:

Find a postcode
Track an item
Currency converter
International post guide
Postage assessment calculator

This restyle does not include functionality changes. Therefore the JavaScript files in this folder have been copied from the original auspost.com.au site. They are only included for illustrative purposes only. It is Australia Post's responsibility to update these JavaScript files.

global.src.js will probably need to be deleted, and web apps.src.js will need to be updated to remove the dependencies it has on global.src.js.
